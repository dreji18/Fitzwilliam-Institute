# Shared Data

This folder holds datasets used across multiple courses.

## Conventions

- Files should be in open formats: `.csv`, `.json`, or `.txt`
- Use `snake_case` filenames
- Add a one-line comment at the top of each CSV describing its source and contents
- Do not commit large files (>5 MB) — link to an external source instead
