# Python 101 — From Zero to Proficient

**Institute:** Fitzwilliam Institute  
**Level:** Beginner → Intermediate  
**Duration:** 12 weeks

---

## Course Overview

A hands-on, practical introduction to Python that takes students from zero programming experience to writing confident, readable, real-world Python. By week 12, students will be able to build file-processing scripts, work with APIs, handle errors gracefully, and understand the foundations needed to go further into data science, web development, or automation.

---

## Weekly Schedule

| Week | Topic | Focus |
|------|-------|-------|
| 01 | [Getting Started](./week-01/) | Python setup, print, variables, basic arithmetic |
| 02 | [Data Types & Strings](./week-02/) | str, int, float, bool, type conversion, f-strings |
| 03 | [Control Flow](./week-03/) | if / elif / else, comparison & logical operators |
| 04 | [Loops](./week-04/) | for, while, range(), break, continue |
| 05 | [Functions](./week-05/) | def, parameters, return values, scope |
| 06 | [Lists & Tuples](./week-06/) | Sequences, indexing, slicing, list methods |
| 07 | [Dictionaries & Sets](./week-07/) | Key-value pairs, set operations, nesting |
| 08 | [File I/O](./week-08/) | Reading & writing files, CSV, context managers |
| 09 | [Error Handling](./week-09/) | try/except, raising exceptions, custom errors |
| 10 | [Modules & Packages](./week-10/) | import, standard library, pip, virtual envs |
| 11 | [Object-Oriented Python](./week-11/) | Classes, objects, methods, inheritance |
| 12 | [Capstone Project](./week-12/) | Real-world mini-project combining all concepts |

## Prerequisites

- No prior programming experience needed
- Python 3.10+ installed ([python.org/downloads](https://www.python.org/downloads/))
- A code editor — [VS Code](https://code.visualstudio.com/) recommended
- Run `bash ../../shared/setup/setup.sh` to set up your environment

---

## How to Use This Repo (Students)

1. Go to the current week's folder
2. Read the `lab/README.md`
3. Open the `lab/` exercise file in your editor
4. Work through exercises top to bottom
5. Only look at `solutions/` after a genuine attempt

## How to Use This Repo (Staff)

Each week follows the same three-folder pattern:
- `lab/` — student-facing starter files with comment scaffolding
- `solutions/` — complete working solutions (staff only)
- `examples/` — live demo scripts used during the lecture

To add a week: copy `../_template/week-01/` and follow the naming convention.
